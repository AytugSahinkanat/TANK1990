# Kuş Bakışı Atış Oyunu (Top-Down Shooter) - Çalıştırma Kılavuzu

## Gereksinimler
- Java JDK 8 veya üzeri
- "images" klasörü (görselleri içeriyor)

## Derleme
Oyunu derlemek için terminalden aşağıdaki komutu çalıştırın:
javac *.java

## Çalıştırma
Derlenen oyunu çalıştırmak için:
java Game

## Kontroller
- WASD: Karakterin hareketi
- Fare: Karakterin hedef alması ve ateş etmesi için
- R: Silahı yeniden doldurma
- 1-5 arası tuşlar: Silah değiştirme (açıldıkça)
- ESC: Oyunu duraklatma/devam ettirme
